
## ✅ 6강: 비교 연산자 & 조건문 (if / else)

### 🔹 기본 문제 6-1

**나이가 20 이상이면 “성인”, 아니면 “미성년자”를 출력하시오.**

```js
let age = 18;
```

📌 출력 예

```
미성년자
```

---

### 🔸 어려운 문제 6-2

**숫자 하나가 주어질 때**

* 0보다 크면 `"양수"`
* 0이면 `"0"`
* 0보다 작으면 `"음수"`
  를 출력하시오.

```js
let num = -3;
```

📌 조건

* 반드시 `if / else if / else` 사용

## ✅ 7강: 논리 연산자 (&&, ||, !)

### 🔹 기본 문제 7-1

**아이디와 비밀번호가 모두 맞으면 “로그인 성공”을 출력하시오.**

```js
let id = "admin";
let pw = "1234";
```

조건

* id가 `"admin"` 이고
* pw가 `"1234"` 일 때만 성공

---

### 🔸 어려운 문제 7-2

**놀이기구 탑승 조건**

* 나이가 12세 이상 **이거나**
* 키가 140cm 이상이면
  👉 `"탑승 가능"`
  그 외는 `"탑승 불가"`

```js
let age = 10;
let height = 150;
```

📌 반드시 `||` 사용

## ✅ 9강: switch 문

### 🔹 기본 문제 9-1

**요일 숫자가 주어질 때 요일을 출력하시오.**

```js

```

📌 출력

```
수요일
```

📌 조건

* 1 → 월, 2 → 화, 3 → 수 … 7 → 일
* 반드시 `switch` 사용
